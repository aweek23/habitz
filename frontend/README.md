# Habitz frontend

Stack Tailwind CSS avec daisyUI, un plugin HyperUI maison, Preline et Flowbite.

## Scripts
- `npm run dev` : lance Vite sur `http://localhost:5173` avec rechargement à chaud.
- `npm run build` : génère les assets de production dans `public/`.
- `npm run preview` : sert le build généré.

## Installation
1. Installer Node.js 18+.
2. Exécuter `npm install` pour récupérer les dépendances (tailwindcss, daisyUI, Preline, Flowbite, Vite, etc.).
3. Lancer le projet avec `npm run dev`.

Le build génère `public/` ignoré par Git. Inclure les fichiers compilés lors du déploiement PHP.
